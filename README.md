# 🧠 AI Annotation Platform

### 🔗 Live Demo

https://ai-annotation-platform-ft8tbtxyatzpxwfkchwwyp.streamlit.app/

AI Annotation Platform is a Streamlit-based web application designed to streamline dataset labeling and annotation workflows. The platform allows users to upload CSV datasets, review text records, assign sentiment labels, and export annotated data for machine learning and NLP applications.

## Features

* 📂 Upload CSV datasets
* 🤖 AI-assisted sentiment label suggestions
* 📝 Manual annotation workflow
* ✏️ Edit saved annotations
* 🚫 Duplicate annotation prevention
* 📊 Progress tracking
* ⬅️➡️ Previous/Next navigation
* 📦 Export annotated datasets to CSV
* 🌐 Live web deployment

## Installation

```bash
pip install streamlit pandas
```

## Run Locally

```bash
streamlit run app.py
```

## Project Workflow

1. Upload a CSV dataset.
2. Review individual text records.
3. Compare AI-generated label suggestions with human judgment.
4. Assign or edit sentiment labels.
5. Save annotations.
6. Export the final annotated dataset as CSV.
